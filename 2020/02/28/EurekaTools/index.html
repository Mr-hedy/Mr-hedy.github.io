Eureka Tools

Eureka小工具
在基于SpringCloud的项目开发中，办公室的同学会直接将自己本地调试的服务注册到公共开发环境的Eureka。 :joy:
这个小工具可以方便对Eureka中的服务进行管理。
其实Eureka本身已经提供了对应用、实例进行管理的接口，但是没有体现在界面上。
那我们就帮帮他，通过Chrome插件的形式向页面中注入管理接口。😄
